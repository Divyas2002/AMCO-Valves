# **App Name**: AMCO Valves

## Core Features:

- Company Introduction Display: Showcase a concise introduction to AMCO Valves and its mission.
- Dynamic Product Catalog: Present detailed information about various valve products, including specifications like 1 Pc, 2 Pc, 3 Pc | Screwed, Socket Weld, Butt Weld, Flanged | Soft & Metal Seat | Floating & Trunnion Mounted Ball Valves.
- Certificates & Approvals Showcase: Display relevant certification logos and documents demonstrating product compliance and quality.
- Interactive Photo Gallery: An image gallery to feature high-quality product images. (Requires backend storage for image data)
- Contact Inquiry Form: A responsive contact form for users to submit their name, email, phone number, and messages, with backend handling for submissions.
- AI-Powered Product Assistant Tool: An intelligent tool that can provide quick answers and detailed information about AMCO Valve products based on natural language queries, summarizing product specifications or usage scenarios.

## Style Guidelines:

- Color scheme is light, prioritizing readability and a modern, professional feel. Primary UI elements use a deep, authoritative blue (#09193d), symbolizing trust and engineering. The background is a very light, desaturated blue (#EBEFF5), providing a clean, airy canvas. A vibrant orange (#f15f23) is used as an accent for call-to-actions and interactive elements, adding dynamism and visual interest. Body text is set in a dark grey (#333) for optimal readability against the light background.
- Headlines and body text utilize 'Inter', a versatile sans-serif typeface, offering a modern, clean, and objective aesthetic that is highly readable across various screen sizes.
- Modern, crisp, line-art icons should be used to convey information clearly and concisely, aligning with the industrial yet clean design language. Icons should support navigation and enhance readability in product details and contact sections.
- The layout will be a single-page, vertically scrolling design optimized for responsiveness across desktops, tablets, and mobile devices. Sections will be clearly demarcated, utilizing a modular grid system to ensure visual balance and hierarchy.
- Subtle, non-intrusive animations will enhance user experience, including smooth scroll effects between sections, subtle hover states for interactive elements, and fading transitions for image galleries to create a fluid interaction.
